POETRY

A Dream On Lease

April 14, 2023


A distant memory keeps calling me,

it keeps reminding me,

about a dream on lease

a dream of seas.


Seas meeting at one point,

not one, two or three,

but millions and countless

each one representing me


Each one colored differently,

each one having differences,

each one contradicting the other,

each one filling the same container.


But I can’t imagine the dimensions

dimensions of the container,

still I am sure it isn’t huge

it is small albeit can’t be filled


A distant memory keeps calling me,

It keeps reminding me,

about a dream on lease,

A dream trying to appease.

~Divyanshu Sharma
